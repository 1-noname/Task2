<h2>Описание</h2>

Приложение включает в себя страницу с авторизацией, списком товаров и возможность фильтровать их по категориям или по названию, а так же детальную информацию о каждом продукте.
Все это включает работу с dummyJson, пользователь для авторизации берутся тут, так же информация о товаре.

В данном приложении реализована защита путей (Пока пользователь не авторизуется у него отсутствует доступ к другим страницам, кроме страницы авторизации)
В качестве стейт менеджера использовался RTK, а для запросов и кэширования RTKQ.
Выбранной архитектурой проекта была FSD
Так же проект включает в себя использование TypeScript для типизации и предотвращения ошибок на этапе разработки
В тестовом задании я старался дробить на чанки большие участки кода, для масштабируемости и переиспользования

Для пользователей с ролью = admin, расширенный функционал, который включает в себя:
  1. Создание новых товаров
  2. Удаление существующих товаров
  3. Редактирование подробной информации о товаре
